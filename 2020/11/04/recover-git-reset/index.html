救回 git reset --hard

前言

前言

先說結論，隨時 git add 以及 git commit，是非常重要的~(慘痛經驗)

事情發生在六角 JS實戰作品班的第二次修改，當時我花了10幾天沒日沒夜終於趕上截止日前一天做完修改(第一版有 git commit 過)，最後打算使用 git，大致記錄一下自己在第二版分別寫了啥功能，不知為何腦袋空空(可能是終於做完了，整個鬆懈)，按了 git reset –hard，按完之後，真的瞬間放空….，開始狂查網路有沒有解決方法，看到有人說連 git add 都沒，沒救了，10幾天沒日沒夜的做，就這樣就沒了，欲哭無淚，還好最後找到方法，雖然有些還是要用手打，但大概復原90%以上，我已經非常滿足，最後多花半天復原。

以下寫下我當時查到的一些方法

我這次的專案是用 vuecli 做的，dist 並未放在 git 追蹤，且有 build 過

查看目前處於什麼狀況

救回前 先看自己是處於什麼狀況

  1. 有 git stash 過
  2. 已 git commit 過
  3. 有 git add 但未 git commit
  4. 未 git add 未 git commit

解法

1. 有 git stash 過

2. 已 git commit 過

3. 有 git add 但未 git commit

4. 未 git add 未 git commit

我是遇到第4個
